Breath.


保持呼吸，你便能够活下去；连活着都变得困难的事，永远也扼制不住生的希望。面对的敌人强大无比，为了战胜它只得拼尽全力。可有时拼尽了全力却也无法获得胜利，但是啊，不要放弃啊。他们从未有过放弃的想法，他们从不为了自己的渺小而放弃斗志；他们为自己，能够活下去、能够生存下去的目标而奋斗努力着。这些愤恨于自己弱小的人聚集在一起，他们围绕着那为首的男人，举起自己的手伸向天空。他们站了起来，他们呐喊了起来，他们的声音穿过了重重阻碍到达了我们的怀中。


他们是，闪耀星天丝吉亚联合会。
